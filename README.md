# Quote-App

This project was generated with [Angular CLI(https://github.com/angular/angular-cli) version 6.2.1.

## Author 
Musyoki Mutua



## DESCRIPTION

This is a simple application using Angular framework that allows you to add quotes.

## User Story

- User can generate a new quote via a form.

- User can delete a generated quote

- Users can see the date the quote was submited.

- Users can see the day count from when the quote was posted.

- User can view Quotes generated using a Quotes API

# TECHNOLOGIES USED

1. HTML

2. Type Script

3. Javascript

4. MarkDown

5. CSS
